# Программа для парсинга всей истории телеграм-канала

Чтобы спарсить телеграм-канал он должен быть публичным

Перед запуском необходимо сделать:
```bash
cp ./env.example ./.env
```

И указать данные от телеграм-апи для дальнейшего прсинга. Получить токен и хэш приложения можно по 
[ссылке](https://my.telegram.org/apps)

На системе должен быть установлен докер, через ENV переменную OUTPUT_DIR можно кастомизировать папку, в которую будут помещены данные. Чтобы задать канал, который надо спарсить необходимо установить значение CHANNEL_NAME как ник телеграм-канала(без @)

Использование:
```bash
make run
```
