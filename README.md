# Программа для парсинга всей истории телеграм-канала

Перед запуском необходимо сделать:
```bash
cp ./env.example ./.env
```

И указать данные от телеграм-апи для дальнейшего прсинга. Получить токен и хэш приложения можно по 
[ссылке](https://my.telegram.org/apps)

На системе должен быть установлен докер, через ENV переменную OUTPUT_DIR можно кастомизировать папку, в которую будут помещены данные
парсинга

Использование:
```bash
make run
```
